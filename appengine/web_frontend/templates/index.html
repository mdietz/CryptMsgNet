How it really works

CryptWeb is a secure messaging application based on PGP that's designed to be as hassle free and secure as possible. Public keys are (and should) only be traded in person. To make this easy, CryptWeb facilitates this process through the use of NFC or QR codes between mobile devices. Communications are still passed through and mediated by a central server, but the server is never able to see the public or private keys of the communicators. This way, privacy can still be maintained if the server is compromised.

Like any technology, CryptWeb can be used for good or evil. Please use for good. Evil makes us sad.
